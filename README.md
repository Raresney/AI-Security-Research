# 🤖 AI Hacking Lab

Research lab focused on exploring the intersection between **Artificial Intelligence** and **Cybersecurity**.

The goal is to understand how AI systems can be **manipulated, abused, or used defensively** in security contexts.

---

## 🧪 Projects

### 🧠 Prompt Injection & AI Manipulation

Experiments related to manipulating AI systems.

**Focus areas**

- Prompt injection attacks
- Instruction override techniques
- AI behavior manipulation

**Goal**  
Understand **AI vulnerabilities** and explore potential **alignment weaknesses**.

---

### 🔎 Recon AI

AI-assisted reconnaissance and target analysis.

**Input**

- `nmap`
- `whois`
- DNS records
- Open ports

**Output**

- Potential attack vectors
- Risk scoring
- Structured summaries

Purpose: transform raw reconnaissance data into **actionable intelligence**.

---

### 🎣 Phishing AI

Research on AI-generated phishing simulations.

**Features**

- AI-generated phishing templates (Email / SMS)
- Social engineering simulations
- Detection experiments

**Classification**

- Phishing
- Legitimate communication

Goal: improve **phishing detection and awareness systems**.

---

## 🛠 Tech Stack

- Python
- AI / LLM tools
- OSINT tools
- Networking utilities (`nmap`, DNS analysis)
- Data analysis

## ⚠️ Disclaimer

This project is created **strictly for research and educational purposes**.

🚫 No illegal activities

The objective is to **study AI security risks and improve defensive systems**.
